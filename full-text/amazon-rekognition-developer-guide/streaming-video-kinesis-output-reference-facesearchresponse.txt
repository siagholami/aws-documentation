FaceSearchResponse
Information about a face detected in a streaming video frame and the faces in a collection that match the detected face. You specify the collection in a call to CreateStreamProcessor. For more information, see Working with Streaming Videos. 
DetectedFace
Face details for a face detected in an analyzed video frame.
Type: DetectedFace object
MatchedFaces
An array of face details for faces in a collection that matches the face detected in DetectedFace.
Type: MatchedFace object array
