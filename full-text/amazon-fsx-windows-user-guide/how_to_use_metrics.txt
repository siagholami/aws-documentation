How to Use Amazon FSx for Windows File Server Metrics
The metrics reported by Amazon FSx provide information that you can analyze in different ways. The list following shows some common uses for the metrics. These are suggestions to get you started, not a comprehensive list.



How Do I Determine...
Relevant Metrics




My file system's throughput?
SUM(DataReadBytes  DataWriteBytes)/Period (in seconds)


My file system's IOPS?
Total IOPS = SUM(DataReadOperations  DataWriteOperations  MetadataOperations)/Period (in seconds)
