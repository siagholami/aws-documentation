NICE DCV Clients
NICE DCV offers a Windows client, Linux client, web browser client, and macOS client. The clients offer similar feature sets, but there are some differences. Choose the NICE DCV client that best suits your needs.

Requirements
For a good user experience with NICE DCV, ensure that the client computers meet the following minimum requirements. Keep in mind that your experience is largely dependent on the number of pixels streamed from the NICE DCV server to the NICE DCV client.

Software
The Native Windows client is supported on 32-bit and 64-bit versions of the following operating systems: [See the AWS documentation website for more details] The client also requires the following additional software: [See the AWS documentation website for more details]
The web browser client is supported on the following browsers across all desktop operating systems: [See the AWS documentation website for more details] The web browser client also requires WebGL and asm.js. The web browser client is not supported on mobile operating systems, such as Android and iOS.
The Linux client is supported on the following modern Linux operating systems: [See the AWS documentation website for more details]
The macOS client requires macOS High Sierra or later.


Network
The client must be able to connect to the NICE DCV server, and it must be able to communicate over the required port (8443 by default).
