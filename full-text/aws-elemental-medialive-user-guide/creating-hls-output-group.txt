Creating an HLS Output Group
Follow these steps if, when you were planning the channel, you determined that you want to include an HLS output group.
To create an HLS output group


On the Create channel page, in the Output groups section, choose Add. The content pane changes to show the Add output group section. 


Choose HLS, and then choose Confirm. More sections appear. 


Complete the fields as described in Fields for the HLS Group. 


After you enter all the information for one output group, you can optionally create another output group. Otherwise, go to the next step.
