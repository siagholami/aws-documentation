Deleting a Robot Application Version
Delete robot application versions you no longer need.
To delete a robot application version
Follow the steps under one of these tabs.

[ Using the console ]


Sign in to the AWS RoboMaker console at [https://console\.aws\.amazon\.com/robomaker/](https://console.aws.amazon.com/robomaker/) .


In the left navigation pane, choose Development, then choose Robot applications.


Choose the Name of the robot application to see its versions.


In the robot detail page, choose the Version to see version details.


In the robot application version details page, choose Delete, and then choose Delete to confirm.



[ Using the AWS CLI ]
Example
Here's an example AWS CLI command that performs the equivalent of the console-based steps.  
$ aws robomaker delete-robot-application--version --application my-robot-application-arn --version 1.5
