NetworkInterface
Describes a network interface.
Contents
networkInterfaceId 
The ID of the network interface.
Type: String
Required: No
privateIpAddress 
The IPv4 address of the network interface within the subnet.
Type: String
Required: No
publicIpAddress 
The IPv4 public address of the network interface.
Type: String
Required: No
