RobotSoftwareSuite
Information about a robot software suite (ROS distribution).
Contents

Type: String
Valid Values:ROS | ROS2

The version of the robot software suite (ROS distribution).
Type: String
Valid Values:Kinetic | Melodic | Dashing 
Required: No
