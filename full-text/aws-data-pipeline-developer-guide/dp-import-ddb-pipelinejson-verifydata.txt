Step 5: Verify the Data Import
Next, verify that the data import occurred successfully using the DynamoDB console to inspect the data in the table.
To verify the DynamoDB table


Open the DynamoDB console.


On the Tables screen, click your DynamoDB table and click Explore Table.


On the Browse Items tab, columns that correspond to the data input file should display, such as Id, Price, ProductCategory, as shown in the following screen. This indicates that the import operation from the file to the DynamoDB table occurred successfully.
