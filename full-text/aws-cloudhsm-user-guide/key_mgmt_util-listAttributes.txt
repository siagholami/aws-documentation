listAttributes
The listAttributes command in key_mgmt_util lists the attributes of an AWS CloudHSM key and the constants that represent them. You use these constants to identify the attributes in getAttribute and setAttribute commands. For help interpreting the key attributes, see the Key Attribute Reference.
Before you run any key_mgmt_util command, you must start key_mgmt_util and log in to the HSM as a crypto user (CU). 
Syntax
This command has no parameters.
listAttributes
Example
This command lists the key attributes that you can get and change in key_mgmt_util and the constants that represent them. For help interpreting the key attributes, see the Key Attribute Reference. 
To represent all attributes in the getAttribute command in key_mgmt_util, use 512.
