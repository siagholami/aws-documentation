Create a Virtual Private Cloud (VPC)
If you don't already have a virtual private cloud (VPC), create one now.
To create a VPC


Open the Amazon VPC console at https://console.aws.amazon.com/vpc/.


On the navigation bar, use the region selector to choose one of the AWS Regions where AWS CloudHSM is currently supported.


Choose Start VPC Wizard.


Choose the first option, VPC with a Single Public Subnet. Then choose Select.


For VPC name:, type an identifiable name such as CloudHSM. For Subnet name:, type an identifiable name such as CloudHSM public subnet. Leave all other options set to their defaults. Then choose Create VPC. After the VPC is created, choose OK.
