Removing tags from an AWS Elemental MediaConvert resource
The following procedure shows you how to remove tags from existing job templates, output presets, and queues using the AWS Elemental MediaConvert console.
To do this using the API, see the PUT method in the Tags endpoint section of the MediaConvert API Reference.
To remove tags from a job template, output preset, or queue (console)


Open the MediaConvert console at https://console.aws.amazon.com/mediaconvert.


Choose the three-bar icon on the left to access the left navigation pane.


Choose Job templates, Output presets, or Queues.


Choose the name of the specific resource that has tags that you want to change.


Choose the Update, Edit queue, or Update preset button in the upper right. 


Next to any tag that you want to delete, choose Remove.


Choose Save.
