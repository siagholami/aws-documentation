Update or Delete a Git Repository
To update a Git repository


On the Git repositories page, choose the repository you want to update.


On the repository page, choose Edit repository. 


Update Git credentials on the repository page.


To delete a Git repository


On the Git repositories page, choose the repository you want to delete. 


On the repository page, choose all the notebooks that are currently linked to the repository. Choose Unlink notebook.


On the repository page, choose Delete.


Note
To delete the local Git repository from Amazon EMR, you must first unlink any notebooks from this repository. For more information, see Link or Unlink a Git Repository. Deleting a Git repository will not delete any secret created for the repository. You can delete the secret in AWS Secrets Manager.
