Savings Plans Details
The SavingsPlan columns provide details about the ComputeSavingsPlans. SavingsPlan columns are visible if you have purchased Savings Plans. For more information about Savings Plans, see What are Savings Plans? in the Savings Plans User Guide.

savingsPlan/AmortizedUpfrontCommitmentForBillingPeriod

Description: The amount of upfront fee a Savings Plan subscription is costing you for the billing period. The initial upfront payment for All Upfront Savings Plan and Partial Upfront Savings Plan amortized over the current month. For No Upfront Savings Plan, the value is 0.
