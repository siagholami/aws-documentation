Disable a group
When you no longer need a group, you can disable it.
To disable a group


Open the Amazon WorkMail console at https://console.aws.amazon.com/workmail/.


If you require a different AWS Region, change it from the navigation bar. For more information, see Regions and endpoints in the Amazon Web Services General Reference.


For Organizations, choose the name of your organization.


In the navigation pane, choose Groups.


In the list of groups, select the group to disable, and then choose Disable group.


In the Disable group(s) dialog box, choose Disable.


Note
To permanently delete a group, use the DeleteGroup API action for Amazon WorkMail. For more information, see DeleteGroup in the Amazon WorkMail API Reference.
