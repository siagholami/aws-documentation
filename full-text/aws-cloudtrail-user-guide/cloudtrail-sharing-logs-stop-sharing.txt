Stop Sharing CloudTrail Log Files Between AWS Accounts
To stop sharing log files to another AWS account, simply delete the role that you created for that account in Creating a Role. 


Sign in to the AWS Management Console as an IAM user with administrative-level permissions for Account A. 


Navigate to the IAM console.


In the navigation pane, click Roles.


Select the role you want to delete.


Right-click and select Delete Role from the context menu.
