AWS Firewall Manager prerequisites
This topic shows you how to prepare your account to use AWS Firewall Manager. Before you use Firewall Manager for the first time, perform all the following steps in sequence. 
Topics
 Step 1: Join AWS Organizations
 Step 2: Set the AWS Firewall Manager administrator account
 Step 3: Enable AWS Config
You can now configure Firewall Manager to begin protecting your resources. For more information, see Getting started with AWS Firewall Manager AWS WAF policies.
