Set Skybox Stretch
Sets the stretch factor, which affects how the skybox stretches vertically. Stretching the skybox lowers the horizon line.

Inputs




Pin
Type
Description




In
Event
Triggers the node.


Amount
Number
Vertical stretch factor.  A value of 0 does not stretch the skybox. Specify a higher value to lower the horizon line. Default value: 0
