Splitting and Merging Roads

This topic references tools and features that are legacy. If you want to use legacy tools in Lumberyard Editor, disable the CryEntity Removal gem using the Project Configurator or the command line. To learn more about legacy features, see the Lumberyard Legacy Reference.
You can split a road apart or merge two roads together.
To split or merge roads


In the Rollup Bar, on the Objects tab, click Misc, Road.


Under Spline Parameters, click Edit, and do the following for the road in your level:
    To split a road apart, select the desired point and click Split.
    To merge two roads together, select the end point of one road and the start point of another road. Then click Merge.
