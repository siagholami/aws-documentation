Environment Node
You can use the Environment node to set the sun's longitude and latitude in a scene.
To add an Environment node in the Track View


In the Track View, right-click either the sequence (top node) or the Director node in the tree as applicable, and then click Add Environment Node.


For each of the keys listed below, click the applicable key listed under the Environment node.


To position a key, double-click the preferred location on its highlighted row in the timeline. Double-click the green marker, and then under Key Properties, enter a value for Value.


Environment Node Key Properties 



Property
Description




Sun Longitude
Sets the sun's longitude.


Sun Latitude
Sets the sun's latitude.
