Quick Start: Running the Samples Project on iOS Devices
You can use this quick start to learn how to run the Samples Project on your iOS device from a macOS computer.
This topic assumes that you have:
 Xcode installed on your macOS computer
 Lumberyard installed on your macOS computer
 An iOS device connected to your computer with a USB cable
 Lumberyard installed on a Windows computer
 Remote Shader Compiler running on your Windows or macOS computer
For more information about creating iOS apps, see iOS Support.
To run the Samples Project on an iOS device


Use Lumberyard Setup Assistant to install the required third-party software and SDKs for iOS.


On your macOS computer, navigate to the lumberyard_version\dev\Tools\LmbrSetup\Mac\ directory and double-click SetupAssistant.exe.


If you see a Custom Install box, click Customize and then verify that the engine root path is correct.


On the Get Started page, select the following:

Run your game project
Run the Lumberyard Editor and tools
Compile the game code
Compile the engine and asset pipeline
Compile the Lumberyard Editor and tools
Compile for iOS devices



Complete the Lumberyard Setup Assistant wizard. For more information, see Running Lumberyard Setup Assistant.


Configure Asset Processor for iOS.


Use your preferred text editor to open the AssetProcessorPlatformConfig.ini file. You can find this file in the lumberyard_version\dev\ directory.


Remove the preceding semicolon to uncomment ios=enabled.
[Platforms]
  ;pc=enabled
  ;es3=enabled
  ios=enabled
  ;osx_gl=enabled


Save the file.


Launch Asset Processor.


Navigate to the following directory and double-click AssetProcessor.exe:

On Windows: lumberyard_version\dev\Bin64vc141\
On macOS: lumberyard_version\dev\BinMac64\



In Asset Processor, verify that the Status is Idle.



In a command line or Terminal window, do the following:


Change directory to the lumberyard_version\dev\ directory.


Type the following:

On Windows: lmbr_waf.bat configure
On macOS: sh lmbr_waf.sh configure



On your Windows or macOS computer, set the IP address for the remote shader compiler server.


Use your preferred text editor to open the system_ios_ios.cfg file. You can find this file in the lumberyard_version\dev\ directory.


Set r_ShaderCompilerServer to the IP address of your Windows computer that runs the shader compiler.


Save the file.


Build and deploy the code.


Navigate to the lumberyard_version\dev\Solutions\ directory and open the LumberyardiOSSDK.xcodeproj file.


Click Run to build and deploy the code to your iOS device.
