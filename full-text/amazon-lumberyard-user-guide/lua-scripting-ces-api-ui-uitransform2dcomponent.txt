SetAnchors
Sets the anchors.
Syntax
void SetAnchors(UiAnchors anchors, bool adjustOffsets, bool allowPush)
For possible values for UiAnchors, see GetAnchors.




Parameter
Description




adjustOffsets
If true, the offsets are adjusted to keep the rectangle in the same position.


allowPush
Only takes effect if the anchors are invalid. If true, when an anchor is changed to overlap the anchor opposite it, the opposite anchor moves.
