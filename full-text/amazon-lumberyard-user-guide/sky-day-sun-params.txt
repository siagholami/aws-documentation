Setting Sun Parameters and Console Variables
You can define how the sun appears in the daytime sky.
To set sun parameters


In Lumberyard Editor, choose Tools, Other, Time Of Day.


In the Parameters pane, for Sun, adjust the Sun Parameters as needed.


Close the Time of Day editor.


You can define how the sun renders in the daytime sky.
To set console variables for sun rendering behavior


In Lumberyard Editor, in the Console pane, click the icon in the bottom left.


In the Console Variables window, search for and set the following console variables to control the sun's rendering behavior. For more information, see Using the Console Window.
