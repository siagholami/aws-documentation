Get Param Number
Returns a material's numerical parameter value for the specified entity.
Contents
 Inputs
 Outputs

Inputs




Pin
Type
Description




In
Event
Triggers the node.


Source
EntityID
References a specific entity from which events are generated. By default, it references Self, the entity to which the Script Canvas component attaches the current script. You can also select another entity. For more information, see Setting Entity Targets.


Param Name
String
The name of the material parameter to query. For more information, see Finding the Material Parameter Name.


Material ID
Number
If the material owner has a multimaterial, use this parameter to select a specific submaterial. IDs start at 1.

Pin
Type
Description




Out
Event

Number
Number
The queried value. If the parameter is not found, the value is 0.
