dynamodb-pitr-enabled
Checks that point in time recovery (PITR) is enabled for Amazon DynamoDB tables. The rule is NON_COMPLIANT if point in time recovery is not enabled for Amazon DynamoDB tables.
Identifier: DYNAMODB_PITR_ENABLED
Trigger type: Configuration changes
Parameters:
None  
AWS CloudFormation template
