elb-cross-zone-load-balancing-enabled
Checks if cross-zone load balancing is enabled for the Classic Load Balancers (CLBs). This rule is NON_COMPLIANT if cross-zone load balancing is not enabled for a CLB. 
Identifier: ELB_CROSS_ZONE_LOAD_BALANCING_ENABLED
Trigger type: Configuration changes
AWS Region: All supported AWS Regions
Parameters:
None  
AWS CloudFormation template
