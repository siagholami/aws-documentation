Delete an AWS DeepLens Project


Open the AWS DeepLens console at https://console.aws.amazon.com/deeplens/home?region=us-east-1#firstrun/.


In the navigation pane, choose Projects.


On the Projects page, choose the button next to the project that you want to delete.


In the Actions list, choose Delete.


In the Delete project-name dialog box, choose Delete to confirm that you want to delete the project.


Delete an AWS DeepLens Model


Open the AWS DeepLens console at https://console.aws.amazon.com/deeplens/home?region=us-east-1#firstrun/.


In the navigation pane, choose Models.


On the Models page, choose the button next to the model that you want to delete.


In the Action list, choose Delete.


In the Delete model-name dialog box, choose Delete to confirm that you want to delete the model.


Delete an AWS DeepLens Device Profile
To deleting an unused device profile, you deregister the device. For more information, see Deregistering Your AWS DeepLens Device. 
Delete AWS DeepLens Logs from CloudWatch Logs
If you exposed personally identifiable information in the AWS IoT Greengrass user logs that were created in Amazon CloudWatch Logs when you ran your AWS DeepLens project, use the CloudWatch Logs console to delete them. 


Sign in to the AWS Management Console and open the CloudWatch console at https://console.aws.amazon.com/cloudwatch/.


In the navigation pane, choose Logs.


On the Log Groups page, choose the button next to the AWS IoT Greengrass user log that you want to delete. The log begins with /aws/greengrass/Lambda/....


In the Actions list, choose Delete log group.


In the Delete log group dialog box, choose Yes, Delete to confirm that you want to delete the logs.
Note
When the Lambda function for your project runs again, the deleted log group will be recreated.
