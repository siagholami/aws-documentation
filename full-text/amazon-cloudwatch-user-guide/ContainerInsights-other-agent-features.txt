Deploying Other CloudWatch Agent Features in Your Containers
You can deploy additional monitoring features in your containers using the CloudWatch agent. These features include the following:
 Embedded Metric Format— For more information, see Ingesting High-Cardinality Logs and Generating Metrics with CloudWatch Embedded Metric Format.
 StatsD— For more information, see Retrieve Custom Metrics with StatsD .
