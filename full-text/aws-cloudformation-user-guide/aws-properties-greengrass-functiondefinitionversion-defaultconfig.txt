AWS::Greengrass::FunctionDefinitionVersion DefaultConfig
The default configuration that applies to all Lambda functions in the function definition version. Individual Lambda functions can override these settings.
 In an AWS CloudFormation template, DefaultConfig is a property of the  AWS::Greengrass::FunctionDefinitionVersion  resource.
Syntax
To declare this entity in your AWS CloudFormation template, use the following syntax:
JSON
{
  "[Execution](#cfn-greengrass-functiondefinitionversion-defaultconfig-execution)" : Execution
}
YAML
[Execution](#cfn-greengrass-functiondefinitionversion-defaultconfig-execution): 
    Execution
Properties
Execution 
Configuration settings for the Lambda execution environment on the AWS IoT Greengrass core.
Required: Yes
Type: Execution
Update requires: No interruption
