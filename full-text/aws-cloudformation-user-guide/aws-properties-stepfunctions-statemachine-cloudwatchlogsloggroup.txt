AWS::StepFunctions::StateMachine CloudWatchLogsLogGroup
Defines a CloudWatch log group.
Note
For more information see Standard Versus Express Workflows in the AWS Step Functions Developer Guide.
Syntax
To declare this entity in your AWS CloudFormation template, use the following syntax:
JSON
{
  "[LogGroupArn](#cfn-stepfunctions-statemachine-cloudwatchlogsloggroup-loggrouparn)" : String
}
YAML
[LogGroupArn](#cfn-stepfunctions-statemachine-cloudwatchlogsloggroup-loggrouparn): String
Properties
LogGroupArn 
The ARN of the the CloudWatch log group to which you want your logs emitted to. The ARN must end with :* 
Required: No
Type: String
Update requires: No interruption
