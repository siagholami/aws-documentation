AWS::Cognito::UserPool Policies

  "[PasswordPolicy](#cfn-cognito-userpool-policies-passwordpolicy)" : PasswordPolicy
}
YAML
[PasswordPolicy](#cfn-cognito-userpool-policies-passwordpolicy): 
    PasswordPolicy
