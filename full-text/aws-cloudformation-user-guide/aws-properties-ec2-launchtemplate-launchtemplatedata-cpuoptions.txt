AWS::EC2::LaunchTemplate CpuOptions
Specifies the CPU options for an instance.
CpuOptions is a property of AWS::EC2::LaunchTemplate LaunchTemplateData.
Syntax
To declare this entity in your AWS CloudFormation template, use the following syntax:
JSON
{
  "[CoreCount](#cfn-ec2-launchtemplate-launchtemplatedata-cpuoptions-corecount)" : Integer,
  "[ThreadsPerCore](#cfn-ec2-launchtemplate-launchtemplatedata-cpuoptions-threadspercore)" : Integer
}
YAML
[CoreCount](#cfn-ec2-launchtemplate-launchtemplatedata-cpuoptions-corecount): Integer
  [ThreadsPerCore](#cfn-ec2-launchtemplate-launchtemplatedata-cpuoptions-threadspercore): Integer
Properties
CoreCount 
The number of CPU cores for the instance.
Required: No
Type: Integer
Update requires: No interruption
ThreadsPerCore 
The number of threads per CPU core. To disable multithreading for the instance, specify a value of 1. Otherwise, specify the default value of 2.
Required: No
Type: Integer
Update requires: No interruption
