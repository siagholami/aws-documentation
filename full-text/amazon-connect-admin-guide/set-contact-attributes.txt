Contact block: Set contact attributes
Contact flow types

 All flows
Description

Stores key-value pairs as contact attributes.
Contact attributes are accessible by other areas of Amazon Connect, such as the CTRs. 

For more information about how to use contact attributes, see Use Amazon Connect contact attributes.
