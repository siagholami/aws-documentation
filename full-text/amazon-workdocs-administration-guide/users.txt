Inviting and managing Amazon WorkDocs users
You can change user roles, invite, enable, or disable users, and change user settings under Manage Users in the admin control panel in the Amazon WorkDocs web client. You can also promote a user to an administrator. For more information, see Promoting a user to administrator.
To open the admin control panel, in Amazon WorkDocs, under My Account, choose Open admin control panel.
Note
Some admin control panel options differ between cloud directories and connected directories.
