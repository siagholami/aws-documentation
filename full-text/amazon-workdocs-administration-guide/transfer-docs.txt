Transferring document ownership
You can transfer an inactive user's files and folders to an active user. For more information on how to inactivate a user, see Disabling users.
To transfer document ownership


From the admin control panel, under Manage Users, search for the inactive user.


Choose the pencil icon () next to the inactive user's name. 


Select Transfer Document Ownership and type the email address of the active user to whom to transfer the files.


Choose Save Changes.
Warning
This action cannot be undone.
