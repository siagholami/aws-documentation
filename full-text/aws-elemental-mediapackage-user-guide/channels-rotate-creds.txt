Rotating Credentials on an Input URL
Rotate credentials on an input URL to generate a new WebDAV user name and password.
You can use the AWS Elemental MediaPackage console or the MediaPackage API to rotate credentials. For information about rotating credentials through the MediaPackage API, see the AWS Elemental MediaPackage API Reference.
To rotate credentials (console)


Open the MediaPackage console at https://console.aws.amazon.com/mediapackage/.


If the Channels page doesn't appear, on the AWS Elemental MediaPackage home page, choose Skip and go to console.


On the Channels page, choose the name of the channel that holds the input URL that you're rotating the credentials for.


On the channel's details page, choose the input URL that you're rotating credentials for, and then choose Rotate credentials.


To confirm that you want to generate a new user name and password, choose Rotate.


AWS Elemental MediaPackage displays the new credentials.
