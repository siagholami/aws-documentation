Amazon EC2 and Tools for Windows PowerShell
You can perform common tasks related to Amazon EC2 using the AWS Tools for PowerShell.
The example commands shown here assume that you have set default credentials and a default region for your PowerShell session. Therefore, we don't include credentials or region when we invoke the cmdlets. For more information, see Getting Started with the AWS Tools for Windows PowerShell.
