# Presto Functions in Amazon Athena<a name="presto-functions"></a>

The Athena query engine is based on [Presto 0\.172](https://prestodb.io/docs/0.172/index.html)\. For more information about these functions, see [Presto 0\.172 Functions and Operators](https://prestodb.io/docs/0.172/functions.html) and the specific sections from Presto documentation referenced below\.

Athena does not support all of Presto's features\. For information, see [Considerations and Limitations](other-notable-limitations.md)\.
