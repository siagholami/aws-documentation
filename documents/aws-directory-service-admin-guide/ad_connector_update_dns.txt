Update the DNS Address for Your AD Connector
Use the following steps to update the DNS addresses that your AD Connector is pointing to.
Note
If you have an update in progress, you must wait until it is complete before submitting another update.
Note
If you are using Amazon WorkSpaces with your AD Connector, ensure that your WorkSpaceâ€™s DNS addresses are updated as well. For more information, see Update DNS Servers for WorkSpaces.
To update your DNS settings for AD Connector


In the AWS Directory Service console navigation pane, choose Directories.


Choose the directory ID link for your directory.


On the Directory details page, choose Network & security. 


In the Existing DNS settings section, choose Update.


In the Update existing DNS addresses dialog, type the updated DNS IP addresses, and then choose Update.
