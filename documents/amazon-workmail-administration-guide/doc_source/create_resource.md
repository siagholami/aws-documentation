# Creating a resource<a name="create_resource"></a>

You can add a new resource to your organization, and allow it to be reserved\.

**To add a resource**

1. Open the Amazon WorkMail console at [https://console\.aws\.amazon\.com/workmail/](https://console.aws.amazon.com/workmail/)\.

1. If necessary, change the AWS Region\. From the navigation bar, choose the Region that meets your needs\. For more information, see [Regions and endpoints](http://docs.aws.amazon.com/general/latest/gr/index.html?rande.html) in the *Amazon Web Services General Reference*\.

1. For **Organizations**, choose the name of your organization\.

1. In the navigation pane, choose **Resources** and **Add resource**\.

1. On the **Add resource details** page, enter values for the **Resource name**, **Description**, **Resource type**, and **Email address** fields\.

1. Choose **Create**\.
