Creating an email flow rule
To create an email flow rule, you specify a rule action to apply to an email message when a specified pattern is matched. When you create a new email flow rule, it's applied immediately.
To create a new email flow rule


Open the Amazon WorkMail console at https://console.aws.amazon.com/workmail/.


For Organizations, choose the name of your organization.


In the navigation pane, choose Organization settings, Inbound/Outbound rules.


Choose Create Rule.


Enter a name for your rule.


Enter one or more sender patterns and one or more recipient patterns.


Select the action to apply to the email.


Choose Create Rule.


You can test the new email flow rule that you created. For more information, see Testing an email flow rule.
