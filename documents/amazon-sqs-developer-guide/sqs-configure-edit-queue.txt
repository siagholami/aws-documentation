Editing an Amazon SQS queue (console)
You can edit all of the queue configuration parameters (except the queue type) and add or remove queue features. 
To edit an Amazon SQS queue (console)


Open the Amazon SQS console at https://console.aws.amazon.com/sqs/.


In the navigation pane, choose Queues. 


From the queue list, choose a queue and choose Edit.


(Optional) Update the queue's Configuration parameters. 


(Optional) Define or update an Access policy.


(Optional) Add, update, or remove Encryption.


(Optional) Add, update, or remove a Dead-letter queue (which allows you to receive undeliverable messages).


(Optional) Add, update, or remove the Tags for the queue.


When you finish configuring the queue, choose Save.


The console displays the Details page for the queue.
