# Editing an Amazon SQS queue \(console\)<a name="sqs-configure-edit-queue"></a>

You can edit all of the queue configuration parameters \(except the queue type\) and add or remove queue features\. 

**To edit an Amazon SQS queue \(console\)**

1. Open the Amazon SQS console at [https://console\.aws\.amazon\.com/sqs/](https://console.aws.amazon.com/sqs/)\.

1. In the navigation pane, choose **Queues**\. 

1. From the queue list, choose a queue and choose **Edit**\.

1. \(Optional\) Update the queue's [Configuration](sqs-configure-queue-parameters.md) parameters\. 

1. \(Optional\) Define or update an [Access policy](sqs-configure-add-permissions.md)\.

1. \(Optional\) Add, update, or remove [Encryption](sqs-configure-sse-existing-queue.md)\.

1. \(Optional\) Add, update, or remove a [Dead\-letter queue](sqs-configure-dead-letter-queue.md) \(which allows you to receive undeliverable messages\)\.

1. \(Optional\) Add, update, or remove the [Tags](sqs-configure-tag-queue.md) for the queue\.

1. When you finish configuring the queue, choose **Save**\.

   The console displays the **Details** page for the queue\.
