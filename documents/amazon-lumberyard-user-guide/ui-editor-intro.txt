Creating and Customizing Project User Interfaces
You can use the UI Editor to create and customize various parts of the game user interface, such as images, text, buttons, menus, scroll boxes, and heads-up displays (HUDs). For a tutorial about UI creation for games, see Lumberyard Tutorials.
