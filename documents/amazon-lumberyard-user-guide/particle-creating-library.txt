Adding Particle Libraries
Particle effect data is stored in an XML-based library file.
To add a particle library


In the Particle Editor, do one of the following:
    Choose Create new library.
    Choose File, Add library.
    Choose the drop-down arrow in the Libraries panel, and choose Add library.
    Choose the  icon to the right of the search bar.



In Libraries, where the default name is highlighted, enter a name for the library.


Choose Add Particle or Add Folder and then enter a name for the new particle or folder.
