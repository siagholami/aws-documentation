Comment Node
Use the Comment node to add comments to your track view sequence. This is mostly used for production purposes and is not rendered in the game.
To add a Comment node in the Track View


In the Track View, right-click either the sequence (top node) or the Director node in the tree as applicable, and then click Add Comment Node.


For each of the keys listed below, click the applicable key listed under the Comment node. 


To position a key, double-click the preferred location on its highlighted row in the timeline. Double-click the green marker, and then under Key Properties enter a value for Value.
