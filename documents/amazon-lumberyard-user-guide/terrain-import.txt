Importing and Exporting Terrain Blocks
You can import and export terrain areas or blocks. When importing or exporting, you should also import or export the associated terrain texture layers.
To import a terrain block and texture layers


In Lumberyard Editor, click Game, Terrain, Import Terrain Block and select a .trb file to import.


Click Terrain, Terrain Texture Layers.


Click File, Import Layers and select terrain texture files to import.


To export a terrain block and texture layers


In Lumberyard Editor, click Game, Terrain, Export Terrain Block, and select a .trb file to export.


Click Terrain, Terrain Texture Layers.


Click File, Export Layers, and select terrain texture files to export.
