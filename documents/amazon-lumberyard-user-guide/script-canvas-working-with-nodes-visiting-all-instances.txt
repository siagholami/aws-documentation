Navigating Graphs
If you want to visit all nodes of the same type on a graph – for example, to check their configuration – use the following procedure.
To cycle through instances of the same node


In the Node Palette search box, enter the name of the node.


In the list of results, click the name of the node whose instances you want to find.


To scroll to the right, starting at the leftmost node occurrence, press F8. The relevant nodes are highlighted with an orange glow.


To scroll to the left, starting at the rightmost node occurence, press F7.


To return to the beginning or end of the cycle, press F7 or F8 again.
