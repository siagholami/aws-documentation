Exporting a Terrain Heightmap
You can export a heightmap file that was created in the Terrain Editor to the following file formats:
 .asc (32-bit)
 .bt (32-bit)
 .tif (32-bit)
 .pgm (16-bit)
 .raw (16-bit)
 .r16 (16-bit)
 .bmp (8-bit)
 .png (8-bit)
To export a heightmap


In Lumberyard Editor, choose Game, Terrain, Edit Terrain.


In Terrain Editor, choose File, Export Heightmap and enter a file name and directory location.
