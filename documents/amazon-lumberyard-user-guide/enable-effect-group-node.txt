Enable Effect Group
Enables a specific group of effects setting that are defined in an xml file. 
Note
You must use the Apply Effect Group At Position node for effect group xml files that include the fadeDistance parameter. 
To disable the effect, see Disable Effect Group.
For more information about effect group files, see Customizing Post-Processing Effects.
