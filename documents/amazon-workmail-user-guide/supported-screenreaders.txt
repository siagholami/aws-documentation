Supported screen readers
You can use the following browser-based screen readers with Amazon WorkMail:
 Firefox - NVDA
 Safari - VoiceOver
Note
For NVDA, some actions work best when you enter or exit Forms mode.
Most of the commands provided can be used across all screen readers. Some screen readers, such as VoiceOver, offer keyboard-based combinations for voiceover controls. For example, in VoiceOver use VORight arrow in place of the right-arrow key on your keyboard.
Note
The following exceptions apply only to VoiceOver: 
VOSpacebar replaces Enter.
VOLeft/Right arrows replace Tab and ShiftTab.
FnDelete replaces Delete.
For best accessibility, use plain text as the email format and disable the Preview pane. For information, see the Email settings section in Working with general and email settings.
