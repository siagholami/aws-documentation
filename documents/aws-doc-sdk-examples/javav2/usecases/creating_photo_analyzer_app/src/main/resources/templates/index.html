AWS Photo Analyzer Application

The AWS Photo Analyzer application is a sample application that uses the Amazon Rekognition Service as well as other AWS Services and the Java V2 SDK. Analyzing nature photographs has never been easier! Simply perform these steps:

  1. You can upload a nature photograph to a S3 bucket by choosing the Upload Photos menu item.
  2. Choose Choose File and browse to a nature image located on your local desktop.
  3. Choose Upload to upload your image to a S3 bucket.
  4. You can choose Get Images to view the images located in the S3 bucket. All images in the S3 bucket are displayed in the table.
  5. You can analyze the photographs and produce a report by choosing the Analyze Photos menu item.
  6. Enter an email address in the email field and choose Analyze Photos.
  7. The Amazon Simple Email Service is used to send an email with an Excel report to the specified email recipient.
