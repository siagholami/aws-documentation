AWS Sample Message Application

The AWS Sample Messaging application is a sample application that uses the Simple Queue Service as well as other AWS Services and the Java V2 API. Working with messages has never been easier! Simply perform these steps:

  1. You can send a new message by choosing the Send Messages menu item. Select a user from the form, enter a message and then choose Send.
  2. The AWS Message application stores the message in a First in First Out (FIFO) queue. This queue ensure that the order of the messages are consisent.
  3. The AWS Message application polls the queue for all messages in the FIFO queue.
  4. The AWS Message application displays the message data in the view. The message body, user name, and an avatar is displayd.
  5. You can send and view multiple messages by using the AWS Message application.
  6. You can purge messages from the queue.
