General Settings Fields
Provide general settings that apply to the entire packaging configuration.


For ID, enter a name that describes the configuration. The ID is the primary identifier for the configuration, and must be unique for your account in the Region.


For Package type, choose Microsoft Smooth.


(Optional) For Segment duration, enter the duration (in seconds) of each segment. If the value that you enter is different from the input segment size, AWS Elemental MediaPackage rounds segments to the nearest multiple of the input segment duration.
