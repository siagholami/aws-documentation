Deleting an Asset
To remove the packaging group URLs and to stop AWS Elemental MediaPackage from delivering further content, delete an asset. 
To delete an asset, you can use the AWS Elemental MediaPackage console, the AWS CLI, or the MediaPackage API. For information about creating a packaging configuration with the AWS CLI or MediaPackage API, see Assets id in the AWS Elemental MediaPackage VOD API Reference.
To delete an asset (console)


Open the MediaPackage console at https://console.aws.amazon.com/mediapackage/.


In the navigation pane, under Video on demand, choose Assets.


On the Assets page, choose the asset using one the following methods: 
    Choose the asset name
    Select the check box next to the asset ID


Choose Delete.
