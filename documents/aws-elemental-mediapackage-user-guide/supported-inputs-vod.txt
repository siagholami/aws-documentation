VOD Supported Codecs and Input Types
The following sections describe supported input types and codecs for file-based video on demand (VOD) content.
Supported Input Types
These are the input types that AWS Elemental MediaPackage supports for VOD content.



MediaPackage Input Type
Use Case




HLS
Pull an HLS stream set from an Amazon Simple Storage Service bucket, with or without a secure connection.Additional requirements:[See the AWS documentation website for more details]


SMIL
Pull an MP4 stream set referenced by a .smil manifest from an Amazon Simple Storage Service bucket, with or without a secure connection. For information about the .smil manifest, see Creating a SMIL File.Additional requirements:[See the AWS documentation website for more details]
