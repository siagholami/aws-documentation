How Amazon Transcribe Medical Works
Amazon Transcribe Medical enables you to transcribe individual audio content with medical information into text. You can use either the streaming transcription API or the batch transcription API. Streaming transcription enables you to transcribe real-time streams of audio into text. To transcribe recorded audio files, use batch transcription. For more information on transcribing streaming audio, see Streaming Transcription Overview and Establish a Bi-Directional Connection Using the WebSocket Protocol. For more information on batch transcription, see Batch Transcription Overview
