KinesisVideoStream
Kinesis video stream stream that provides the source streaming video for a Amazon Rekognition Video stream processor. For more information, see CreateStreamProcessor.
Contents
Arn 
ARN of the Kinesis video stream stream that streams the source video.
Type: String
Pattern: (^arn:([a-z\d-]):kinesisvideo:([a-z\d-]):\d{12}:.$) 
Required: No
