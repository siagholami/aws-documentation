KinesisDataStream
The Kinesis data stream Amazon Rekognition to which the analysis results of a Amazon Rekognition stream processor are streamed. For more information, see CreateStreamProcessor.
Contents
Arn 
ARN of the output Amazon Kinesis Data Streams stream.
Type: String
Pattern: (^arn:([a-z\d-]):kinesis:([a-z\d-]):\d{12}:.$) 
Required: No
