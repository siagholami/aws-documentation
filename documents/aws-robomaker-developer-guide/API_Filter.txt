Filter
Information about a filter.
Contents
name 
The name of the filter.
Type: String
Length Constraints: Minimum length of 1. Maximum length of 255.
Pattern: [a-zA-Z0-9_\-]* 
Required: No
values 
A list of values.
Type: Array of strings
Array Members: Fixed number of 1 item.
Length Constraints: Minimum length of 1. Maximum length of 255.
Pattern: [a-zA-Z0-9_\-]* 
Required: No
