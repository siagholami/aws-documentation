# Supported Compression and Archive File Formats<a name="macie-compression-archive-formats"></a>

Currently, Macie supports the following compression and archive file formats:
+ BZIP
+ GZIP
+ LZO
+ RAR
+ SNAPPY
+ AR
+ CPIO
+ Unix dump
+ TAR
+ zip
+ XZ
+ Pack200
+ BZIP2
+ 7z
+ ARJ
+ LZMA
+ DEFLATE
+ Brotli
