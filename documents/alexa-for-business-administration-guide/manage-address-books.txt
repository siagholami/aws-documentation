Managing Address Books
To call contacts from your shared devices, you must first create an address book and assign it to the room profile associated with the rooms where the devices are assigned. You can create multiple address books, but you can only assign one address book to a room profile.
To create an address book


Open the Alexa for Business console at https://console.aws.amazon.com/a4b/.


Choose Calls, Create address book, and enter a unique name for the address book.


To assign an address book to a room profile


Open the Alexa for Business console at https://console.aws.amazon.com/a4b/.


Choose Rooms profiles and select the name of the room profile to edit.


Under Outbound calling, select the address book to assign, and then choose Save.


To edit an address book


Open the Alexa for Business console at https://console.aws.amazon.com/a4b/.


Choose Calls and select the name of the address book to edit.


Edit the values for Name and Description, and then choose Save.


To delete an address book


Open the Alexa for Business console at https://console.aws.amazon.com/a4b/.


Choose Calls and select the name of the address book to delete.


Choose Delete address book, Delete.
