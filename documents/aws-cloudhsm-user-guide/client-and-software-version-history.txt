AWS CloudHSM Client and Software Information
To manage and use the HSMs in your cluster, you use the AWS CloudHSM client and related software libraries. If you installed the AWS CloudHSM client for Linux or Windows and any required software libraries, you have all the software needed to use AWS CloudHSM. 
This section provides information about supported platforms and a full version history.
