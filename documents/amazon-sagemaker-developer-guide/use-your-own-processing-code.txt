Use Your Own Processing Code
You can install libraries to run your scripts in your own processing container or, in a more advanced scenario, you can build your own processing container that satisfies the contract to run in Amazon SageMaker. For a formal specification that defines the contract for an an Amazon SageMaker Processing container, see Build Your Own Processing Container (Advanced Scenario).
