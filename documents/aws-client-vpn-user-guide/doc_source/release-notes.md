## Supported OpenVPN directives<a name="openvpn-directives"></a>

The AWS\-provided client supports the following OpenVPN directives:
+ client
+ dev
+ proto
+ remote
+ remote\-random\-hostname
+ resolv\-retry
+ nobind
+ persist\-key
+ persist\-tun
+ remote\-cert\-tls
+ cipher
+ verb
+ ca
+ reneg\-sec
+ cert
+ key
+ auth\-user\-pass
+ connect\-retry
+ static\-challenge
+ tun\-mtu
+ tun\-mtu\-extra
