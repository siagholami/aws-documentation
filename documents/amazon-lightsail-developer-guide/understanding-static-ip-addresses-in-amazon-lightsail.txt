Static IP addresses in Amazon Lightsail
Note
A static IP is a fixed, public IP address that you can assign and reassign to an instance or other resource. If you haven't set up a static IP address, each time you stop or restart your instance, Lightsail assigns a new public IP address.
Last updated: March 23, 2017 
Learn about these common tasks for working with static IP addresses:
  Create a static IP address in Lightsail 
  Restart, stop, or start your Lightsail instance 
Important
If you stop or restart your instance without first creating a static IP address, you lose your IP address when your instance restarts. It's always a good idea to create a static IP address for your instance. Here's how: Create a static IP address in Lightsail.
For more information, see Understanding public IP addresses and private IP addresses in Lightsail.
