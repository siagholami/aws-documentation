Make outbound calls
Before you can make an outbound call, your contact center must be set up to allow agents to make calls. For more information, see Step 3: Telephony options in Create an Amazon Connect instance.
For information about the caller ID that's displayed when you make an outbound call, see Set up outbound caller ID.
To make an outbound call


In your Contact Control Panel, choose Number pad.


Enter the number you want to call and press Call.
