Show agent queues in a Queues table
By default agent queues don't appear in a Queues table in a historical metrics report. You can choose to show them.
To show agent queues in a Queues table


In a historical metrics report, choose the Settings icon.



Choose Filters, Show agent queues, Agent queues, and then use the drop-down to choose the agent's queues you want to include in the table. 



Choose Apply. The agent queues you selected appear in the Queues table in the historical metrics report.
