AWS::AutoScalingPlans::ScalingPlan TagFilter
TagFilter is a subproperty of ApplicationSource that specifies a tag for an application source to use with AWS Auto Scaling. 
Syntax
To declare this entity in your AWS CloudFormation template, use the following syntax:
JSON
{
  "[Key](#cfn-autoscalingplans-scalingplan-tagfilter-key)" : String,
  "[Values](#cfn-autoscalingplans-scalingplan-tagfilter-values)" : [ String, ... ]
}
YAML
[Key](#cfn-autoscalingplans-scalingplan-tagfilter-key): String
  [Values](#cfn-autoscalingplans-scalingplan-tagfilter-values): 
    - String
Properties
Key 
The tag key.
Required: Yes
Type: String
Minimum: 1
Maximum: 128
Pattern: [\u0020-\uD7FF\uE000-\uFFFD\uD800\uDC00-\uDBFF\uDFFF\r\n\t]*
Update requires: No interruption
Values 
The tag values (0 to 20).
Required: No
Type: List of String
Update requires: No interruption
