AWS::CloudFront::Distribution Restrictions
A complex type that identifies ways in which you want to restrict distribution of your content.
Syntax
To declare this entity in your AWS CloudFormation template, use the following syntax:
JSON
{
  "[GeoRestriction](#cfn-cloudfront-distribution-restrictions-georestriction)" : GeoRestriction
}
YAML
[GeoRestriction](#cfn-cloudfront-distribution-restrictions-georestriction): 
    GeoRestriction
Properties
GeoRestriction 
A complex type that controls the countries in which your content is distributed. CloudFront determines the location of your users using MaxMind GeoIP databases. To disable geo restriction, remove the Restrictions property from your stack template.
Required: Yes
Type: GeoRestriction
Update requires: No interruption
