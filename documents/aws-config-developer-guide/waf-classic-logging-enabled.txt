waf-classic-logging-enabled
Checks if logging is enabled on AWS Web Application Firewall (WAF) classic global web ACLs. This rule is NON_COMPLIANT for a global web ACL, if it does not have logging enabled. 
Identifier: WAF_CLASSIC_LOGGING_ENABLED
Trigger type: Periodic
AWS Region:Only available in US East (N. Virginia) Region
Parameters:
KinesisFirehoseDeliveryStreamArns (Optional)Type: CSV
Comma separated list of Amazon Kinesis stream ARN for AWS WAF logs.
