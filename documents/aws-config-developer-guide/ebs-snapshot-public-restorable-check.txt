ebs-snapshot-public-restorable-check
Checks whether Amazon Elastic Block Store snapshots are not publicly restorable. The rule is NON_COMPLIANT if one or more snapshots with the RestorableByUserIds field is set to all. If this field is set to all, then Amazon EBS snapshots are public.
Identifier: EBS_SNAPSHOT_PUBLIC_RESTORABLE_CHECK
Trigger type: Periodic
AWS Region: All supported AWS Regions except Africa (Cape Town) and Europe (Milan)
Parameters:
None  
AWS CloudFormation template
