atan(Decimal)
Returns the inverse tangent of a number in radians. Decimal arguments are rounded to double precision before function application.
Examples: atan(0) = 0.0



Argument type
Result




Int
Decimal (with double precision), the inverse tangent of the argument. Imaginary results are returned as Undefined.


Decimal
Decimal (with double precision), the inverse tangent of the argument. Imaginary results are returned as Undefined.
