# Math activity operators and functions<a name="math-operators-functions"></a>

You can use the following operators in a `math` activity:


|  |  | 
| --- |--- |
|  \+  |  addition  | 
|  \-  |  subtraction  | 
|  \*  |  multiplication  | 
|  /  |  division  | 
|  %  |  modulo  |
