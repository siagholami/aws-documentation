cos(Decimal)
Returns the cosine of a number in radians. Decimal arguments are rounded to double precision before function application.
Examples: cos(0) = 1



Argument type
Result




Int
Decimal (with double precision), the cosine of the argument. Imaginary results are returned as Undefined.


Decimal
Decimal (with double precision), the cosine of the argument. Imaginary results are returned as Undefined.
