Amazon EKS optimized Bottlerocket AMIs
The Amazon EKS optimized Bottlerocket AMI is built on top of Bottlerocket. The AMI is configured to work with Amazon EKS and it includes containerd and  kubelet .
Select a link in one of the following tables to view the latest Amazon EKS optimized Bottlerocket AMI ID for a region and Kubernetes version. You can also retrieve the IDs with an AWS Systems Manager parameter using different tools. For more information, see Retrieving Amazon EKS optimized Bottlerocket AMI IDs.
