Copying an object
In the Amazon S3 console, you can copy an object to a bucket or a folder. For more information, see Copying objects in the Amazon Simple Storage Service Developer Guide.
To copy an object


Sign in to the AWS Management Console and open the Amazon S3 console at https://console.aws.amazon.com/s3/.


In the Buckets list, choose your bucket name.


On the Overview tab, select the check box beside the object that you want to copy.


Choose Actions, and choose Copy.


Choose the copy destination:
    If you want to copy your object to a bucket, select the bucket.
    If you want to copy your object to a folder in a bucket, choose the bucket name, and then choose the folder.


Under Review, confirm the copy details, and choose Copy.


Amazon S3 copies your object to the destination.
