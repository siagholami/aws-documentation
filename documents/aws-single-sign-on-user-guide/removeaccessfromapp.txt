Remove User Access
Use this procedure to remove user access to cloud applications or custom SAML 2.0 applications.
To remove user access from an application


Open the AWS SSO console.


Choose Applications.


In the list of applications, choose an application whose access you want to remove. 


On the application details page, choose the Assigned users tab. Select the user or group that you want to remove and then choose Remove.


In the Remove access dialog box, verify the user or group name. Then choose Remove access.
