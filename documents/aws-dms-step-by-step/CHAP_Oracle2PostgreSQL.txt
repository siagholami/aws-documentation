Rolling Back the Migration
If there are major issues with the migration that cannot be resolved in a timely manner, you can roll back the migration. These steps assume that you have already prepared for the rollback as described in Step 8: Cut Over to PostgreSQL.
To roll back the migration


Stop all application services on the target PostgreSQL database.


Let the AWS DMS task replicate remaining changes back to the source Oracle database.


Stop the PostgreSQL to Oracle AWS DMS task.


Start all applications back on the source Oracle database.
