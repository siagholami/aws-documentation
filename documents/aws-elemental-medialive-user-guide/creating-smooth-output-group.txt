Creating a Microsoft Smooth Output Group
Follow these steps if, when you were planning the channel, you determined that you want to include a Microsoft Smooth output group.
To create a Microsoft Smooth output group


On the Create channel page, in the Output groups section, choose Add. The Add output group section appears. 


Choose Microsoft Smooth, and then choose Confirm. More sections appear. 


Complete the fields as described in Fields for the Microsoft Smooth Group. 


After you enter all the information for one output group, you can optionally create another output group. Otherwise, go to the next step.
