Integrating an Amazon Lex Bot with Slack
This exercise provides instructions for integrating an Amazon Lex bot with the Slack messaging application. You perform the following steps:


Create an Amazon Lex bot.


Create a Slack messaging application. 


Integrate the Slack application with your bot Amazon Lex.


Test the integration by engaging in conversation with your Amazon Lex bot. You send messages with the Slack application and test in a browser window.
