# FlowerTypes\.json<a name="gs-cli-create-flower-types-json"></a>

The following code is the JSON data required to create the `FlowerTypes` custom slot type:

```
{
    "enumerationValues": [
        {
            "value": "tulips"
        },
        {
            "value": "lilies"
        },
        {
            "value": "roses"
        }
    ],
    "name": "FlowerTypes",
    "description": "Types of flowers to pick up"
}
```
