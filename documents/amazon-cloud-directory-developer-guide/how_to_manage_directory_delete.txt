Delete Your Directory
Use the following procedure to delete a directory in Cloud Directory.
Note
Before you can delete a directory, you must first disable it. For instructions, see Disable Your Directory.
To delete a directory


In the AWS Directory Service console navigation pane, under Cloud Directory, select Directories.


Select the option in the table next to the directory ID you want to delete.


Choose Actions.


Choose Delete


In the Delete directory dialog, confirm the operation by typing in the name of your directory, and then choose Delete.
