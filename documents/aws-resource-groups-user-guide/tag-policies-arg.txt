Tag policies
A https://docs.aws.amazon.com/organizations/latest/userguide/orgs_manage_policies_tag-policies.html is a type of policy that you create in AWS Organizations. You can use tag policies to help standardize tags across resources in your organization's accounts. To use tag policies, AWS recommends that you follow the workflows described in Getting started with tag policies in the AWS Organizations User Guide. As mentioned on that page, the recommended workflows include finding and correcting noncompliant tags. To accomplish these tasks, you use the Resource Groups console.
