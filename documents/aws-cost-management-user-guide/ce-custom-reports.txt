Managing your saved Cost Explorer reports
You can save the results of a Cost Explorer query as a Cost Explorer report. This enables you to track your Cost Explorer results and forecasts over time.

Creating a Cost Explorer report
You can use the console to save the results of a Cost Explorer query as a report.
Note
Cost Explorer reports can be modified. We strongly recommend that you don't use them for auditing purposes.
To save a Cost Explorer report


Sign in to the AWS Management Console and open the Billing and Cost Management console at https://console.aws.amazon.com/billing/home#/.


In the navigation pane, choose Cost Explorer.


On the Cost Explorer page, choose Launch Cost Explorer.
Note
Cost Explorer is available in any AWS account for no cost.


Choose New report. This resets all of your Cost Explorer settings to your default settings. 


For the report name text field, enter a name for your report.


Customize your Cost Explorer settings.


Choose Save report.


In the Save report dialog box, choose Continue.


Viewing a Cost Explorer report
You can use the console to view saved Cost Explorer reports.
To view your saved reports


Sign in to the AWS Management Console and open the Billing and Cost Management console at https://console.aws.amazon.com/billing/home#/.


In the navigation pane, choose Cost Explorer.


On the Cost Explorer page, choose Launch Cost Explorer.
Note
Cost Explorer is available in any AWS account for no cost.


On the report dropdown menu, choose View/Manage all reports.


To return to the Cost Explorer page, choose Back.


Editing a Cost Explorer report
You can use the console to edit Cost Explorer reports.
To edit your report


Sign in to the AWS Management Console and open the Billing and Cost Management console at https://console.aws.amazon.com/billing/home#/.


In the navigation pane, choose Cost Explorer.


On the Cost Explorer page, choose Launch Cost Explorer.
Note
Cost Explorer is available in any AWS account for no cost.


On the report dropdown menu, choose the report that you want to edit.
Note
You can't edit the preconfigured reports. If you choose one of the preconfigured reports as a starting point for a report, enter a new report name in the report name field and continue with this procedure.


Customize your Cost Explorer settings.


Choose Save report.


In the Save report dialog box, choose Continue.


Deleting a Cost Explorer report
You can use the console to delete saved Cost Explorer reports.
To delete a saved report


Sign in to the AWS Management Console and open the Billing and Cost Management console at https://console.aws.amazon.com/billing/home#/.


In the navigation pane, choose Cost Explorer.


On the Cost Explorer page, choose Launch Cost Explorer.
Note
Cost Explorer is available in any AWS account for no cost.


On the report dropdown menu, choose View/Manage all reports.


Next to the report that you want to delete, select the check box.


On the navigation bar, choose Delete.


In the Delete Report dialog box, choose Delete.
