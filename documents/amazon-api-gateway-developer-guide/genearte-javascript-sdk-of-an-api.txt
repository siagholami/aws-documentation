Generate the JavaScript SDK of an API
To generate the JavaScript SDK of an API in API Gateway


Sign in to the API Gateway console at https://console.aws.amazon.com/apigateway.


In the list of APIs, choose the name of the API that you want to generate the SDK for.


In the pane on the left that contains the name of the API, choose Stages. 


In the Stages pane, choose the name of the stage.


On the SDK Generation tab, for Platform, choose JavaScript. 


Choose Generate SDK, and then follow the on-screen directions to download the SDK generated by API Gateway.


Follow the instructions in Use a JavaScript SDK generated by API Gateway for a REST API to use the generated SDK.


Every time you update an API, you must redeploy the API and regenerate the SDK to have the updates included.
